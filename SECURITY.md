# Security Policy

## Reporting Vulnerabilities
Report privately via GitHub Security Advisories or email security@fairness-platform.dev.

## Scope
- FastAPI backend (auth, SSRF via dataset upload, injection)
- Frontend XSS vectors
- Dependency vulnerabilities (Dependabot enabled)

## Response Time
- Acknowledgement: 48 hours
- Fix / mitigation plan: 30 days (critical), 90 days (other)
